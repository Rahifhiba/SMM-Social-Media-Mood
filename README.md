# 😀 SMM: Social Media Mood 🙁
## Description
Web application that allows users to search a topic and check general opinion's sentiment around it (positive, negative and neutral)

## 💿 Installation:
for  nootbook:

```
pip install -r requirements.txt

```
for web application:

```
cd Webapp ;
pip install -r requirements.txt

```

## ⚙️Usage

